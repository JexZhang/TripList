# 行迹 - 旅行攻略管理小程序

> 旅行攻略 + 清单 + 地图 + 天气，一站式旅行规划工具

## 📱 项目简介

**行迹**是一款基于 Taro 框架开发的多端旅行规划小程序，帮助用户管理旅行攻略、记录行程开销、整理行李清单。支持微信小程序、H5、支付宝小程序等多端部署。

### ✨ 核心功能

- **📋 旅行攻略** - 按天规划行程，包含景点、餐饮、住宿安排，实时天气信息
- **💰 行程开销** - 自动统计总花费，支持住宿/餐饮/门票/交通分类管理，人均计算
- **🎒 行李清单** - 分类整理行装，支持模板导入、自定义添加、勾选跟踪
- **🎨 多主题切换** - 提供手紙、杂志、明信片、极简四种视觉主题

## 🛠 技术栈

- **框架**: [Taro 4.2](https://taro.zone/) - 多端统一开发框架
- **UI**: React 18 + TypeScript
- **样式**: Sass
- **构建**: Webpack 5
- **代码规范**: ESLint + Stylelint + Commitlint + Husky

## 📦 快速开始

### 环境要求

- Node.js >= 16
- npm >= 8 或 yarn >= 1.22

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
# 微信小程序
npm run dev:weapp

# H5
npm run dev:h5

# 支付宝小程序
npm run dev:alipay
```

### 构建生产版本

```bash
# 微信小程序
npm run build:weapp

# H5
npm run build:h5

# 其他平台
npm run build:swan        # 百度小程序
npm run build:tt          # 字节跳动小程序
npm run build:qq          # QQ小程序
npm run build:jd          # 京东小程序
npm run build:harmony-hybrid  # 鸿蒙混合应用
```

## 🎯 功能说明

### 1. 旅行攻略（攻）

- **按天浏览** - 横向切换日期，查看每日行程
- **天气信息** - 显示当日温度和天气描述
- **景点卡片** - 时间线展示景点、餐饮、住宿
- **实时编辑** - 可直接修改住宿/餐饮/门票费用

### 2. 行程开销（算）

- **总览统计** - 显示总花费、人均消费
- **可视化分布** - 条形图展示住宿/餐饮/门票/交通占比
- **交通明细** - 列出各段交通方式和费用，支持编辑
- **自动计算** - 修改任意金额自动更新总计

### 3. 行李清单（册）

- **分类管理** - 按类别组织行李项（证件、衣物、电子等）
- **模板导入** - 预设多种场景模板，支持合并/替换模式
- **进度跟踪** - 显示已收拾/总数比例
- **自定义添加** - 随时添加新物品

### 4. 主题系统

- **手紙** - 温暖日式风格
- **杂志** - 现代排版设计
- **明信片** - 复古旅行感
- **极简** - 清爽极简风

## 🔧 配置说明

### 应用配置

在 `src/app.config.ts` 中可修改：
- 导航栏颜色和标题
- 页面路由
- 懒加载策略

### 旅行数据

旅行数据存储在 `src/data/trips.json`，包含：
- 行程基本信息（人数、货币、日期范围）
- 每日详细安排
- 交通段信息

## 🚀 部署

### 微信小程序

1. 运行 `npm run build:weapp`
2. 打开微信开发者工具
3. 导入 `dist` 目录
4. 上传代码并提交审核

### H5 部署

1. 运行 `npm run build:h5`
2. 将 `dist` 目录部署到静态服务器
