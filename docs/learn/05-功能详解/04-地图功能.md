# 第五章 · 地图功能

---

## 地图 Tab 的功能

- 用彩色标记点显示所有地点
- 日期筛选（只看某一天的地点）
- 连线显示行程路线
- 自动调整视野（让所有点都在屏幕里）
- 点击标记查看地点详情

---

## 核心挑战：Map 不能随意重渲染

微信的 `<Map>` 是「原生组件」，渲染成本极高。每次 React 触发重渲染，Map 都会重新初始化，视野会重置。

如果不做优化：
- 打开地点详情弹窗 → 父组件状态变化 → Map 重渲染 → 视野跳回默认位置

解决方案：**SheetContainer 隔离**

---

## SheetContainer：性能优化的关键

```tsx
// src/views/MapView/index.tsx

// ① 把弹窗状态封装在独立组件里
const SheetContainer = forwardRef(function SheetContainer(_props, ref) {
  const [picked, setPicked] = useState<Spot | null>(null)  // 只有这个状态影响弹窗

  useImperativeHandle(ref, () => ({
    show: (spot: Spot) => setPicked(spot),
  }))

  return <SpotMapSheet spot={picked} onClose={() => setPicked(null)} />
})

// ② MapView 只保存对 SheetContainer 的引用
export default function MapView() {
  const sheetRef = useRef<SheetHandle | null>(null)

  // ③ 点击标记时，命令式调用 SheetContainer.show()
  const handleTap = useCallback((e: any) => {
    const id = e.detail?.markerId
    const { dayIdx, spotIdx } = decodeMarkerId(id)
    const spot = trip.days[dayIdx]?.spots[spotIdx]
    if (spot) sheetRef.current?.show(spot)  // ← 直接调用方法，不改 MapView 的状态
  }, [trip.days])

  return (
    <View>
      <Map ... onMarkerTap={handleTap} />  {/* Map 不受弹窗状态影响 */}
      <SheetContainer ref={sheetRef} />    {/* 弹窗独立管理自己的状态 */}
    </View>
  )
}
```

结果：
- 打开弹窗 → 只有 `SheetContainer` 重渲染，Map 保持不动 ✅
- 关闭弹窗 → 同样，Map 不受影响 ✅

## 标记点（Markers）

每个有经纬度的地点都会生成一个标记：

```tsx
const markers = useMemo(() => {
  return located.map((p) => {
    const label = mode === 'all' ? p.dayIdx + 1 : p.spotIdx + 1
    const color = dayColor(p.dayIdx)  // 每天一种颜色

    return {
      id: encodeMarkerId(p.dayIdx, p.spotIdx),  // 编码为数字 ID
      latitude: p.lat,
      longitude: p.lng,
      width: 1,   // 标记图标 1x1（透明），实际显示的是气泡
      height: 1,
      anchor: { x: 0.5, y: 1 },  // 锚点在底部中心
      callout: {
        content: String(label),     // 气泡里的数字
        color: '#FFFFFF',
        bgColor: color,             // 气泡背景色
        fontSize: 14,
        padding: 10,
        borderRadius: 999,          // 圆形气泡
        display: 'ALWAYS',          // 一直显示（不是点击才显示）
        textAlign: 'center',
        anchorY: -4,                // 气泡在图标上方偏移
      },
      iconPath: '',  // 空字符串 = 透明图标
    }
  })
}, [located, mode])
```

### encodeMarkerId / decodeMarkerId

微信 Map 的 marker ID 必须是整数。行迹用简单的数学把「第几天第几个地点」编码成一个数字：

```typescript
// src/views/MapView/helpers.ts
export function encodeMarkerId(dayIdx: number, spotIdx: number): number {
  return dayIdx * 1000 + spotIdx
  // 第0天第0个 → 0
  // 第0天第1个 → 1
  // 第1天第0个 → 1000
  // 第2天第3个 → 2003
}

export function decodeMarkerId(id: number): { dayIdx: number; spotIdx: number } {
  return {
    dayIdx: Math.floor(id / 1000),
    spotIdx: id % 1000
  }
}
```

限制：每天最多 999 个地点（够用了）。

### dayColor：每天一种颜色

```typescript
const COLORS = [
  '#FF7A2E', // 橙
  '#4FB286', // 绿
  '#5B9BD5', // 蓝
  '#FF5B5B', // 红
  '#FFC247', // 黄
  '#9B7FD4', // 紫
  '#5CC8FF', // 天蓝
]

export function dayColor(dayIdx: number): string {
  return COLORS[dayIdx % COLORS.length]  // 循环使用颜色
}
```

## 路线连线（Polyline）

当只看某一天的地点时，按地点顺序连线显示路线：

```tsx
const polyline = useMemo(() => {
  if (mode === 'all' || located.length < 2) return []  // 全览或只有一个点：不显示连线

  return [{
    points: located.map(p => ({ latitude: p.lat, longitude: p.lng })),
    color: dayColor(typeof mode === 'number' ? mode : 0),
    width: 10,
    borderColor: '#FFFFFF',  // 白色描边，让线在地图上更清晰
    borderWidth: 2,
    arrowLine: true,          // 显示箭头方向
    dottedLine: false,
  }]
}, [located, mode])
```

## 自适应视野

让所有地点都出现在屏幕中：

```tsx
useEffect(() => {
  if (!ctxRef.current) ctxRef.current = Taro.createMapContext(MAP_ID)
  const ctx = ctxRef.current

  if (located.length === 0) {
    setManualScale(undefined)
    return
  }
  if (located.length === 1) {
    setManualScale(14)  // 只有一个点：固定缩放级别
    return
  }

  // 多个点：自动计算包含所有点的视野
  setManualScale(undefined)
  ctx.includePoints({
    points: located.map(p => ({ latitude: p.lat, longitude: p.lng })),
    padding: [80, 40, 80, 40],  // 上右下左留边距（px）
  })
}, [located])
```

## ModeBar：三种样式变体

`ModeBar` 是地图上方的「按哪天筛选」控件，行迹实现了三种样式：

```tsx
// src/views/MapView/ModeBar.tsx
export default function ModeBar({ days, mode, variant, onChange }) {
  if (variant === 'track') return <TrackBar ... />     // 颜色胶囊
  if (variant === 'segmented') return <SegmentedBar .../>  // 分段控制器
  return <RouteBar ... />                              // 路线卡片
}
```

三种变体对应三种示例攻略：

| 变体 | 对应攻略 | 特点 |
|------|---------|------|
| `track`（默认）| 南京 | 彩色胶囊，按下高亮 |
| `segmented` | 漠河 | iOS 风格分段控制器 |
| `route` | 九寨沟 | 卡片连线，路线感 |

## collectLocated：收集有坐标的地点

```typescript
// src/views/MapView/helpers.ts
export function collectLocated(days: Day[], onlyDayIdx?: number) {
  const result: Array<{ lat: number; lng: number; dayIdx: number; spotIdx: number }> = []

  days.forEach((day, dayIdx) => {
    if (onlyDayIdx !== undefined && onlyDayIdx !== dayIdx) return  // 筛选某一天

    day.spots.forEach((spot, spotIdx) => {
      if (spot.lat && spot.lng) {  // 只收集有坐标的地点
        result.push({ lat: spot.lat, lng: spot.lng, dayIdx, spotIdx })
      }
    })
  })

  return result
}
```

## SpotMapSheet：地点详情弹窗

点击地图标记，弹出地点详情：

```tsx
// src/views/MapView/SpotMapSheet.tsx
function SpotMapSheet({ spot, onClose }) {
  const visible = !!spot

  return (
    <View className={`mv-sheet-mask ${visible ? 'visible' : ''}`} onClick={onClose}>
      <View className='mv-sheet' catchMove onClick={e => e.stopPropagation()}>
        {/* 顶部拖拽条 */}
        <View className='mv-sheet-indicator' />

        {spot && (
          <>
            <Text className='mv-sheet-tag'>{TYPE_LABELS[spot.type]}</Text>
            <Text className='mv-sheet-name'>{spot.name}</Text>
            {spot.city && <Text className='mv-sheet-sub'>{spot.city}</Text>}
            {spot.note && <Text className='mv-sheet-note'>{spot.note}</Text>}
            {spot.price > 0 && <Text className='mv-sheet-price'>¥{spot.price}</Text>}
          </>
        )}
      </View>
    </View>
  )
}
```

弹窗用 CSS transition 动画，而不是 `animation` keyframe：

```scss
.mv-sheet-mask {
  opacity: 0;
  pointer-events: none;
  transition: opacity 0.22s ease;
}
.mv-sheet-mask.visible {
  opacity: 1;
  pointer-events: auto;
}
.mv-sheet {
  transform: translateY(100%);
  transition: transform 0.36s var(--ease-spring);
}
.mv-sheet-mask.visible .mv-sheet {
  transform: translateY(0);
}
```

这样的好处：切换显示/隐藏时有平滑过渡，而不是突然出现/消失。

## 小结

地图功能的关键设计：

| 问题 | 解决方案 |
|------|---------|
| Map 频繁重渲染 | SheetContainer 隔离弹窗状态 |
| Marker ID 只能是整数 | `encodeMarkerId()` 数学编码 |
| 多天颜色区分 | `dayColor()` 7色循环 |
| 视野自适应 | `ctx.includePoints()` |
| 弹窗动画 | CSS transition |
