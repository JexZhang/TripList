## 项目简介

**行迹**是一款微信小程序，帮助用户规划和管理旅行攻略。主要功能：
- 📋 **攻略**：按天管理行程地点
- 💰 **开销**：记录并统计旅行花费
- 🎒 **清单**：出发前的打包核查清单
- 🗺️ **地图**：可视化显示所有地点
- 👥 **协作**：邀请朋友共同编辑
- 🔗 **分享**：生成分享链接（只读/协作）

---

## 技术栈速览

```
前端框架: Taro 4 + React 18 (TypeScript)
样式: SCSS
状态: React Context + Reducer
后端: 微信云开发 (Cloud Functions + Cloud DB)
地图: 高德地图 API
```

> 💡 **给初学者的建议**：不要试图一次全部读完。先读第一、二章，跑起来项目看看效果，再慢慢深入。
