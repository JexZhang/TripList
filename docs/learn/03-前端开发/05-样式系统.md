# 第三章 · 样式系统

## SCSS：CSS 的超集

行册用 SCSS（Sassy CSS）写样式。SCSS 在 CSS 基础上增加了很多功能，但最终编译成普通 CSS。

```scss
// CSS 原来的写法（嵌套时很繁琐）
.spot-card { background: white; }
.spot-card .sc-name { font-size: 30rpx; }
.spot-card .sc-name:active { opacity: 0.7; }

// SCSS 写法（嵌套更清晰）
.spot-card {
  background: white;

  .sc-name {          // 自动处理成 .spot-card .sc-name
    font-size: 30rpx;

    &:active {        // & 代表父选择器：.spot-card .sc-name:active
      opacity: 0.7;
    }
  }
}
```

## 设计 Token 系统

行册在 `src/app.scss` 里定义了一套「设计变量」（CSS 自定义属性），所有颜色、间距、阴影统一管理：

```scss
page {
  /* ===== 颜色系统 ===== */
  --bg: #FFF4E6;          // 页面背景（暖橘米色）
  --bg-soft: #FFE9D1;     // 略深的背景
  --surface: #FFFFFF;     // 卡片背景（白）
  --surface-2: #FFF9F1;   // 次要卡片背景（暖白）
  --ink: #2B1A10;         // 主文字色（深棕）
  --ink-2: #5A4338;       // 次要文字（中棕）
  --ink-3: #9B8378;       // 辅助文字（浅棕）

  --accent: #FF7A2E;      // 品牌橙色（主按钮、强调）
  --accent-2: #FF9A4D;    // 渐变橙色
  --accent-soft: #FFD3A8; // 淡橙（边框、分割线）
  --accent-bg: #FFEBD6;   // 极淡橙（标签背景）
  --coral: #FF5B5B;       // 红色（危险操作、价格）
  --leaf: #4FB286;        // 绿色（成功状态）

  --line: rgba(43, 26, 16, 0.08);   // 分割线（极淡）
  --line-2: rgba(43, 26, 16, 0.14); // 分割线（略深）

  /* ===== 阴影 ===== */
  --shadow-sm: 0 4rpx 14rpx rgba(255, 122, 46, 0.10);
  --shadow-md: 0 12rpx 32rpx rgba(58, 36, 25, 0.10);
  --shadow-lg: 0 24rpx 48rpx rgba(255, 122, 46, 0.18);

  /* ===== 圆角 ===== */
  --r-xs: 12rpx;      // 极小圆角（标签）
  --r-sm: 16rpx;      // 小圆角（输入框）
  --r-md: 22rpx;      // 中圆角（卡片）
  --r-lg: 32rpx;      // 大圆角（区域）
  --r-xl: 40rpx;      // 超大圆角（Sheet 顶部）
  --r-pill: 999rpx;   // 胶囊形（Tab、徽标）
}
```

使用时直接 `var(--变量名)`：

```scss
.spot-card {
  background: var(--surface);    // 白色卡片背景
  border-radius: var(--r-md);    // 中圆角
  box-shadow: var(--shadow-sm);  // 小阴影
  color: var(--ink);             // 深棕文字
}

.es-delete {
  color: var(--coral);           // 红色（危险操作）
  background: rgba(255, 91, 91, 0.08);
}
```

**好处**：改一处，全局都变。比如想改主题颜色，只修改 `--accent` 的值就行。

## 字体系统

行册有三种字体，营造「杂志风格」：

```scss
/* 在 page {} 中定义 */
--font-serif: "Noto Serif SC", "Songti SC", serif;  /* 衬线体：标题、品牌名 */
--font-sans: "PingFang SC", "Noto Sans SC", sans-serif;  /* 无衬线：正文 */
--font-mono: "JetBrains Mono", "SF Mono", monospace;  /* 等宽体：时间、价格、日期 */
```

使用场景：
```scss
.th-name {
  font-family: var(--font-serif);  // 攻略标题用衬线体，更有质感
}

.sc-price {
  font-family: var(--font-mono);   // 价格用等宽体，数字对齐美观
}
```

## Flex 布局（最重要的布局方式）

Flex 是现代 CSS 最常用的布局方式，行册几乎所有布局都用它。

```scss
/* 基础：横向排列 */
.row {
  display: flex;
  align-items: center;  /* 垂直居中 */
}

/* 纵向排列 */
.column {
  display: flex;
  flex-direction: column;
}

/* 两端对齐 */
.space-between {
  display: flex;
  justify-content: space-between;
}

/* 子元素占满剩余空间 */
.flex-child {
  flex: 1;
}
```

行册里的典型 flex 使用：

```scss
/* 攻略详情页头部：名称 + 菜单按钮 */
.th-row {
  display: flex;
  align-items: center;
  justify-content: space-between;  /* 名称在左，按钮在右 */
}

/* Tab 导航栏：均分宽度 */
.trip-tabs {
  display: flex;
}
.tt-item {
  flex: 1;  /* 每个 tab 平均分配宽度 */
  text-align: center;
}
```

## position 定位

```scss
/* 固定在屏幕上（不随滚动） */
.edit-spot-mask {
  position: fixed;
  inset: 0;  /* 等于 top:0; right:0; bottom:0; left:0; */
  /* 覆盖整个屏幕 */
}

/* 相对于父元素定位 */
.spot-card {
  position: relative;  /* 建立定位基准 */

  &::before {
    position: absolute;  /* 相对于 .spot-card 定位 */
    left: 0;
    top: 0;
    bottom: 0;
    width: 6rpx;         /* 左侧橙色竖线 */
    background: var(--accent);
  }
}
```

`z-index` 控制层叠顺序（谁在最上面）：

```scss
.mv-sheet-mask { z-index: 1000; }  /* 地图弹窗：最高层 */
.edit-spot-mask { z-index: 90; }   /* 编辑弹窗 */
.spot-search-mask { z-index: 100; } /* 搜索弹窗（在编辑弹窗之上）*/
```

## 动画系统

行册在 `app.scss` 里定义了全局 keyframes：

```scss
/* 弹窗从下往上滑入 */
@keyframes sheet-up {
  from { transform: translateY(100%); }
  to   { transform: translateY(0); }
}

/* 淡入 */
@keyframes fade-in {
  from { opacity: 0; }
  to   { opacity: 1; }
}

/* 页面进入动画 */
@keyframes page-in {
  from { opacity: 0; }
  to   { opacity: 1; }
}

/* 卡片向上滑入 */
@keyframes slide-up {
  from { opacity: 0; transform: translateY(40rpx); }
  to   { opacity: 1; transform: translateY(0); }
}
```

使用动画：

```scss
.edit-spot-sheet {
  animation: sheet-up 0.4s var(--ease-spring) both;
  /* 动画名  时长   缓动函数             填充模式 */
}
```

缓动函数（控制动画节奏）：

```scss
--ease-out: cubic-bezier(0.22, 0.61, 0.36, 1);      /* 减速（自然停下）*/
--ease-in-out: cubic-bezier(0.65, 0, 0.35, 1);      /* 先加速后减速 */
--ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1);   /* 弹性（超过一点再回来）*/
```

## 渐变按钮

行册所有主要按钮都用橙色渐变：

```scss
.es-save {
  background: linear-gradient(
    135deg,
    var(--accent) 0%,    /* 左上角：#FF7A2E */
    var(--accent-2) 100% /* 右下角：#FF9A4D */
  );
  color: #FFF;
  box-shadow: 0 12rpx 28rpx rgba(255, 122, 46, 0.40);
}
```

标题文字渐变（黑色到橙色）：

```scss
.th-name {
  background: linear-gradient(135deg, var(--ink) 0%, var(--accent) 130%);
  -webkit-background-clip: text;
  background-clip: text;
  color: transparent;  /* 颜色透明，露出背景渐变 */
}
```

## BEM 命名规范

行册的 CSS 类名遵循简化的 BEM（Block-Element-Modifier）规范：

```
ss-input    → spot-search（块）- input（元素）
ss-result   → spot-search（块）- result（元素）
ss-result-name → spot-search（块）- result（元素）- name（子元素）
tt-item     → trip-tabs（块）- item（元素）
tt-item.on  → .on 是修饰符（激活状态）
```

实际代码：
```scss
.spot-search-sheet { /* 弹窗整体 */ }
  .ss-head { /* 头部 */ }
    .ss-title { /* 标题 */ }
    .ss-close { /* 关闭按钮 */ }
  .ss-input { /* 输入框 */ }
  .ss-results { /* 结果列表 */ }
    .ss-result { /* 每一条结果 */ }
      .ss-result-name { /* 地点名 */ }
      .ss-result-addr { /* 地址 */ }
  .ss-manual { /* 手动添加按钮 */ }
```

这种命名方式让你一眼知道某个类属于哪个组件，也避免了类名冲突。

## rpx：响应式像素

小程序专用单位，按屏幕宽度自动缩放：

```
屏幕宽 375px（iPhone SE）：1rpx = 0.5px
屏幕宽 390px（iPhone 14）：1rpx = 0.52px
屏幕宽 414px（iPhone Plus）：1rpx = 0.552px
```

设计稿通常按 750px 宽度设计，设计稿上 50px = 代码里 `50rpx`，直接对应，不需要换算。

## 小结

| 技术 | 用途 | 行册里的例子 |
|------|------|------------|
| CSS 变量 | 设计 Token | `--accent`, `--surface`, `--r-md` |
| SCSS 嵌套 | 清晰的层级 | `.spot-card { .sc-name { ... } }` |
| Flex | 布局 | 几乎所有布局都用 flex |
| position fixed | 浮层/弹窗 | 所有 Sheet 和遮罩 |
| keyframes | 动画 | `sheet-up`, `fade-in`, `slide-up` |
| rpx | 响应式尺寸 | 所有尺寸都用 rpx |
